# Lancet Colour Palette

A 15-colour palette inspired by *The Lancet* journal style, suitable for
categorical data visualisation.

## Usage

``` r
pal_lancet
```

## Format

A `palette` object (character vector of 15 hex colour codes). Printing
in the console displays coloured swatches automatically.
